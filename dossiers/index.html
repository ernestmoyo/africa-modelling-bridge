Dossiers

Build-ready research packages, country-led.

A dossier turns a vague topic into a build-ready package — question, background, data sources, methods landscape, starter pack, success criteria, take-up pathway, supervisor. Full dossiers ship on v0.1. Previews are live now.

What is a dossier?

Eight sections. One question.

Every dossier is built to the same anatomy. A funder can read the question and the budget ask in 90 seconds. A student can read the methods landscape and the starter pack in an afternoon and start the work the same week.

  1. 01

    The Question

    A single sentence a programme manager would take into a meeting.

  2. 02

    Background

    Why this question matters now, in this country, on this timeline.

  3. 03

    Data sources

    Public datasets — HMIS, DHS, IDSR, NMCP bulletins — fully named.

  4. 04

    Methods landscape

    The plausible methods, with the trade-offs and the right-sized choice.

  5. 05

    Starter pack

    Code, data extracts, notebooks — enough to get the student to week 4.

  6. 06

    Success criteria

    What a finished thesis or output actually has to show.

  7. 07

    How to take it up

    BSc, MSc or PhD framing; supervisor channels; stipend mechanics.

  8. 08

    Supervision + adviser

    Named local supervisor; named pro-bono methods adviser.
