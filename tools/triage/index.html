01

Public-domain only

Every document AMB processes is already public. The platform makes it findable, citable and teachable — never republishes anything confidential. Hash-checked against a registry of sources MoHs have asked us not to ingest.

02

Editor review is mandatory

Outputs are AI-generated drafts. From v0.5 onwards, every generated question goes into an editor queue. Ernest, country anchors and working-modeller advisers review before anything publishes as a dossier.

03

The cost is small and visible

A single triage run costs about USD 0.06 in Anthropic API tokens. From v0.5 the public analytics dashboard shows month-to-date AI spend per module — funders see exactly what their tokens bought.
